USO DE FORMULAS

Para la definicin de: cargas de fuerza (Forces), de presin (Pressure Forces), rotaciones (Rotations) y condiciones de contorno (Boundary Conditions) pueden usarse formulas. Estas deben estar escritas en lenguaje Python (3.4) y pueden hacer uso de la librera math.

Las variables de posicin que pueden ser llamadas son X, Y y Z (en maysculas). Estas variables corresponden a las coordenadas de posicin, respecto el sistema de referencia global, de los nodos, elementos, aristas de elementos o caras de elementos a los que se define una carga o condicin.

Sobre condiciones de contorno o fuerzas nodales las variables X, Y y Z hacen referencia a las coordenadas sobre los nodos que recaen.

Cuando se definen cargas sobre superficies o lneas, la posicin (X,Y,Z) hace referencia al centro geomtrico de las caras o aristas de elementos que discretizan las superficies o lneas seleccionadas. Cuando se definen rotaciones sobre elementos usando formulas las variables X, Y y Z hacen referencia al centro geomtrico de los elementos.

Para evitar errores de calculo al dividir por alguna de las variables de posicin, X, Y o Z sern sustituidas por 0.00000001 cuando su valor sea cero.

Operadores principales que pueden ser usados:

Operator

Description

Example

+ Addition

Adds values on either side of the operator.

10 + 20 = 30

- Subtraction

Funciones Trigonomtricas principales:

Operator

Description

math.cos(x)

Return the cosine of x radians.

math.acos(x)

Return the arc cosine of x, in radians.

math.sin(x)

Return the sine of x radians.

math.asin(x)

Return the arc sine of x, in radians.

math.tan(x)

Return the tangent of x radians.

math.atan(x)

Return the arc tangent of x, in radians.

math.atan2(y,x)

Return atan(y / x), in radians. The result is between -pi and pi. The vector in the plane from the origin to point (x, y) makes this angle with the positive X axis. The point of atan2() is that the signs of both inputs are known to it, so it can compute the correct quadrant for the angle. For example, atan(1) and atan2(1, 1) are both pi/4, but atan2(-1, -1) is -3*pi/4.

math.hypot(x,y)

Return the Euclidean norm, sqrt(x*x + y*y). This is the length of the vector from the origin to point (x, y).

math.degrees(x)

Convert angle x from radians to degrees.

math.radians(x)

Convert angle x from degrees to radians.

math.pi

The mathematical constant π = 3.141592, to available precision.

math.e

The mathematical constant e = 2.718281, to available precision.

Ejemplo 1 ya definido de rotacin sobre elementos: Y tangent to Z

En este primer ejemplo se define una rotacin sobre elementos con el fin de orientar el eje local Y perpendicularmente al eje global Z

La formula que se usa en el primer ngulo de rotacin de Euler (sobre Z) ser:

Euler Z: math.degrees(math.atan2(Y,X))

Euler X: 0.0

Euler Z: 0.0

La funcin math.degrees convierte un valor de radianes a grados. La funcin math.atan2 es una funcin trigonomtrica que dado Y y X da el angulo Beta acotado entre -pi y +pi.
